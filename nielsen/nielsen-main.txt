カール・ニルセン　交響曲第4番　作品29　『消しがたきもの』（1914-1916）


チャイコフスキーやサン=サーンスなどと比べると，カール・ニルセン[1]と聞いて代表作がパッと浮かぶ方はきっと多くはないだろう．
今から160年前の1865年6月3日，カール・アウグスト・ニルセンはデンマークの集落ソルテルングで生を受けた．童話作家アンデルセンが生まれたオーデンセに程近いこの土地はフュン島のなだらかな丘陵地帯にあり，まわりを野原や畑に囲まれた大地を感じられる土地である．幼いカール自身も仕事の手伝いや徒弟奉公をしなければならないほど貧しかった一家であったが，父はフィドルやコルネット，母は声楽の腕があり，音楽に触れる機会の多い環境だった．
「以前に音楽を聴いたこと，父が演奏するヴァイオリンやコルネットを聴いたこと，母が歌うのを聴いたこと，そしてはしかで寝ていた時，小さなヴァイオリンに乗せて自分を外へ連れ出そうとしたこと」．ニルセンは音楽との出会いをこのように語っている．


ニルセンが作曲家として活動していたのは19世紀末から20世紀初頭にかけてあるが，その頃は同い年かつ同じ北欧フィンランド生まれのジャン・シベリウス（1865-1957）が交響曲や管弦楽曲で活躍し，グスタフ・マーラー（1860-1911）がウィーンで絶頂と転落を迎えた時期とも重なる．当初から国際的な注目を浴びていた彼らと違い，



はじめの頃のニルセンはヨーロッパはおろかデンマーク国内でも傍流止まりであった．早くから注目を集めたシベリウスの影に隠れてしまったのもあるだろう．だが別の理由として，当時のデンマークはまだ保守的な志向の聴衆が多く，後述のように非伝統的な調性や音楽様式などを多用したニルセンの音楽を受け入れられなかったとする見方も強い．また，それはニルセンが卒業したコペンハーゲン音楽院（のちのデンマーク王立音楽アカデミー）でも同じで，「現代音楽について何かを知りたいと思ったら，ニルス・ゲーデ（当時の音楽院長）から学べるとは期待できない」とニルセンが記憶するほど保守的な風潮が音楽院全体に満ちていた．その後，『愛の賛歌』（1897年初演）や『仮面舞踏会』（1906年初演）でようやく国内の支持を集め，ようやく世界への躍進を遂げたのは第二次大戦後のことである．

同世代の作曲家に比べるとややマイナーではあるが，現代では独特な曲想で多くの聴衆の心を掴んでいる．しかし，ニルセンの当時は独特であるがゆえに保守的なデンマーク国民には受け容れがたい部分もあったようだ．


// ニルセンの前半生　かるーーーく
今から160年前の1865年6月3日，カール・アウグスト・ニルセンはデンマークにある村ソルテルングで生を受けた．童話作家アンデルセンの出身地であるオーデンセに程近いこの村はのフュン島のなだらかな丘陵地帯にあり，まわりを野原と畑に囲まれた大地を感じられる土地である．幼いカール自身も仕事の手伝いや徒弟奉公をしなければならないほど貧しかった一家であったが，父はフィドルやコルネット，母は声楽の腕があり，音楽に触れる機会の多い環境だった．ニルセンは音楽との出会いを次のように説明している．「以前に音楽を聴いたこと，父が演奏するヴァイオリンやコルネットを聴いたこと，母が歌うのを聴いたこと，そしてはしかで寝ていた時，小さなヴァイオリンに乗せて自分を外へ連れ出そうとしたこと．」
また，ニルセンが6歳ではしかに罹ったとき，母親から壁にかかっていた3/4ヴァイオリンを渡された．彼は独学で母が歌ったいくつかのメロディーを弾き，帰宅した父親に演奏した．静かにそれを聴いた父親はヴァイオリンを受け取り，調律して息子に返してやった，という家族の記録も残されている．
ニルセンの両親は彼が音楽家として将来生きていけるとは考えなかったが，オーデンセ第16大隊の軍楽隊に空きができたと聞くと，ニルセンはトランペットを集中して練習し，オーディションを受け，合格することができた．悪い仲間や「悪い」女性との危険な交際について母親から警告を受けて1879年に部隊に着任してからも，彼はヴァイオリンを諦めることはなかった．本格的に音楽を学ぼうと決意したニルセンはコペンハーゲン音楽院を目指すことを決め，古くから親しんできたヴァイオリンで受験したが不合格となってしまった．そこで音楽院長であるゲーゼにこれまでに書いた作品を見せたところ，作曲を学ぶべく入学することを許された．
卒業してからはデンマーク王立管弦楽団の第2ヴァイオリンとして働きつつ，多くの作品を生み出していった．また，奨学金を得て留学旅行へ行ったことも会った．その先で出会った

// 副題の解釈
この交響曲の副題は今回，一般によく知られている『不滅』ではなく『消しがたきもの』という訳をとっている．

この交響曲そのものが滅びることのないように，という意味ではない

「（前略）新しい作品のアイデアがあります．プログラムはありませんが，私たちが『生命の衝動』あるいは『生命の表現』と呼んでいるもの，つまり動くもの，生きる意志を持つもの，善とか悪とか，高いとか低いとか，大きいとか小さいとか，そういう言葉で表現できるものではなく，ただ『生命であるもの』あるいは『生きる意志を持つもの』です．『壮大』とか『繊細で優美』とか，温かいとか冷たいとか（もしかしたら激しいとか）とか，そういう特別な概念ではなく，ただ生命と動き，それぞれが異なり，非常に異なるものの，ある文脈の中で，ある種絶えず流れ，一つの大きな流れの中にあるのです．このことを表す短いタイトルを一言で表したいのです．それで十分でしょう．自分が何を求めているのかをうまく説明することはできませんが，私が求めているのは良いものです．考えると，それが私の中に流れ込んでいるのを感じますが，言葉ではどうにもなりません．」

「生の力，生きんとする消すことのできぬ意志」
"Det Uudslukkelige"→直訳：「消えることのないもの」


第1部   Allegro
{譜例1}
前打音付きの力強いトゥッティで曲が始まると同時に，複数の主題群が同時に進行していく（第1部 Allegro，ヘ長調）．一瞬聴くだけだとかなり複雑怪奇な音楽に感じるかもしれないが，同じリズムが音を変えていたり，ある主題が別の主題に対応する動きを持っていたりなど，けして滅茶苦茶なわけではないことがわかる．やや音楽がスローダウンして木管楽器のさえずりが聞こえると，ファゴットに乗せられてクラリネットが第2主題を提示する{譜例2}．このテーマは終局まで何度も形を変えて登場し，交響曲全体を統一する循環主題のように有機的なつながりを持つ役割を果たしている．その後高音楽器による第2主題と低音楽器群の三連符のモティーフに移行するが，これはニルセンが生涯，強い想い入れを持ったフュン島の風や打ち付ける波のうねりなのかもしれない．
risoluto e giusto（決然と，正確な速度で）で音楽が前向きに動くと金管楽器が順に第2主題を重ねてゆく．ここで1つ目の頂点が形作られるが，高揚感はすぐに冷めてしまう．1番フルートと4番ホルンというかけ離れたパートによるブリッジを挟むと展開部へ入り，前半ではヴァイオリンの跳ね回るような動きに対してヴィオラが不意打ちをかけるように介入してくる．そこへ3番ファゴットや低弦，さらに1番ティンパニが加わって緊張が加速すると，全楽器による激しいトゥッティとなる．ここも冒頭のような複雑な主題構成ではあるが，幻想を見るかのようなソリが奏でられると次第に動きはまとまり，オスティナート風の分散和音と持続低音に支えられた第2主題がやや形を変えて歌われる．
次第に空気は熱を帯びていき，冒頭の主題がトゥッティが回帰する再現部では，第2主題の断片が何度も繰り返されるうちに再び頂点へ到達する．




第2部   Poco Allegretto
昂ったエネルギーが次第に落ち着いていくと，ティンパニのサラバンドやヴァイオリンに導かれて第2部（Poco Allegretto，ト長調）へと移り変わる．ほぼ全ての金管楽器・打楽器を休みにし，木管楽器が主体となって奏されるはデンマークの田舎や田園風景を想起させる．主部ではクラリネットやフルートを中心として素朴な音色が響き，朗らかな中間部の旋律やマンドリン風のピツィカートは少年時代に演奏したダンスの伴奏だろうか．フルート，クラリネット，ファゴット，チェロと受け継がれる伸びやかなソロとピツィカート風の動きが交差すると，ファゴットの幕引きによって再び主部へ回帰する．やがて時間は流れていき，1日は静かな終わりへと向かう．

第3部   Poco Adagio quasi Andante
// 田園風景のような前半部を打ち壊す導入
しかし日常は突如暗転し，悲痛な叫びが穏やかな風景を無きものにする（第3部，Poco Adagio quasi Andante，嬰ハ短調）｛譜例｝．低弦やティンパニの戦禍を仄めかす動機に脅かされながら，ヴァイオリンの哀歌はヴィオラ，チェロへと受け継がれ，高弦と対位法的に絡み合う．嘆きは次第に静まるが再び弦楽器が祈りを捧げるかのように歌い出し，そこへ木管の下降動機が何度も降り注ぐ．だがその祈りも，豹変した木管楽器の強襲で破られてしまう{譜例}．これもまた歌のように聴こえる警告句は転調しながら再登場し，聖歌のような荘厳さを保ちながらフーガへ発展していく．弦楽器はユニゾンへ融合し，だんだんと空気が張り詰めていくにつれてホルンとトロンボーンが祈りの歌をまた出現させる．金管の警告句，弦・木管の下降動機がひとつのクライマックスを作り上げるとまた落ち着きを取り戻し，祈りや哀歌がコラールで再現される．このような歌のようなフレーズには，ニルセンが近い時期に出版した「デンマーク歌曲集」の影響を感じさせる．
オーボエが思い出したかのようにもう一度歌い始めるが，それが次第に緊張となり，con anima（魂を込めて）とともにヴァイオリン，次いで中低弦が急速なパッセージで動き出す．弦楽器が下降してホルンが加わると，満を持して登場する2番ティンパニの打撃が新たな予感と緊張をもたらす．一瞬の静寂ののち，そのままの勢いで第4部（Allegro，ホ長調）へと突入する．



第4部   Allegro
// 2番ティンパニが火蓋を切り落とす
エネルギーが解放されてトゥッティで飛び出していくと力強く主題が提示される｛譜例｝が，これはしばしば「不滅の主題」とも称される．しかしすぐに破壊的なリズムと不協和音によって打ち砕かれ，フガートを挟んでさらに凶暴さを増すと1番ティンパニが先導して2番ティンパニとの戦いが始まる．ニールセンが交響曲のこの脅迫的な要素を終盤まで遅らせたのには深い理由があり，このように進化の過程のある段階でのみ現出する鋭く対立する支配的な力を精密に描写しようとしたためと考えられている．これによりこの交響曲を19世紀的な意味での「標題音楽」の枠を超えたものにしている．既出主題が飛び交いつつ他の楽器も巻き込みながら戦禍は広がってゆくが，オルガンのようなgloriosoが輝かしく示される．
テンションが落ち着いてからは威厳的な性格を保ったティンパニ群のロールを背景に展開され，弦楽器のフガートは真冬のような静けさを感じさせてくれる．内なるエネルギーが高まりつつも，静謐なコラールが中間部を締めくくる．
コントラファゴットとティンパニ群によって音楽が再始動すると，今度は2番ティンパニの先導で争いへともう一度雪崩込む．激しい応酬が他の楽器をよそに繰り広げられ，ズタズタに断片化された第1部第2主題がそこへ入り込む．中低弦，そして高弦のフガートが緊張感を強めていき，コーダでようやく第1部第2主題が完全に回帰し，調性（ホ長調）も確立される．2番ティンパニが「生命の力」を高らかに宣言するとテーマは歌い上げられ，音楽は徐々に，力強くフィナーレへと向かう．








// 1938年の演奏会プログラムに記載された，ニルセンが残したとされる解説
「音楽は生命である．
たとえ一音でも空気や空間に響き渡れば，それは生命と動きの結果である．だからこそ，音楽（そして踊り）は生への意志の最も直接的な表現なのである．
交響曲は，生命の最も根源的な源泉，そして生命感覚の源泉，すなわち，私たちが見て，感じ，そして生きるすべての人間，動物，植物の生命の根元にあるものを呼び起こす．それは，限られた時間と空間における生命の発展を，標題のように音楽的に記述するものではなく，むしろ，半ば混沌とし，完全に初歩的な感情の層へと，標題のようにではなく，深く掘り下げていくものである．言い換えれば，標題のように聞こえるにもかかわらず，あらゆる標題音楽とは正反対のものである．
交響曲は，様々なセクションの構成や音楽素材の配列が，技師が堤防や水門を建設するのと同じように作曲家の熟慮の賜物である点を除けば，考える部分を持つものではない．洪水の際の水の流れ．それはある意味で，鳥が鳴き，動物が吠え，鳴き，走り，戦い，そして人間が何の説明もなくうめき，うなり，歓喜し，叫ぶ理由を，全く考えなしに表現したものだ．交響曲はこれらすべてを描写するのではなく，その根底にある根本的な感情を描写する．音楽はまさにそれを可能にする．音楽は音楽の最も深遠な性質であり，その真の領域である…なぜなら，音楽はただそれ自体であることによって，その使命を果たしたからだ．なぜなら，音楽は人生そのものであるが，他の芸術は人生を表現したり言い換えたりするだけである．人生は不屈で，消えることがない．闘争，格闘，創造，そして衰退は，今日も昨日のように，明日も今日のように続き，そしてすべては戻ってくる．
もう一度言うが，音楽は生命であり，生命と同様に消しがたきものである．」


// 初演時のプログラムの要約
「作曲者はこのタイトルを一言で表すことで，音楽のみが最大限に表現し得るもの，すなわち生命の根源的な意志を示そうとした．音楽は生命であり，生命と同様に消しがたきものである．したがって，作曲者がこの音楽作品に与えたタイトルは，余計なものに思えるかもしれない．しかし，作曲者はこの語を用いて，この主題の音楽的性格を強調したのである．これは標題ではなく，音楽の本来の領域に関する示唆的なヒントに過ぎない．」


ニルセンは標題音楽としてではなく
